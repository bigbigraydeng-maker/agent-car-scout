## 🎯 改进方案

### 1. ✅ 已售状态检测模块

**文件**: `vehicle-status-check.js`

**功能**:
- 检测车辆页面是否包含"sold"/"已售"关键词
- 检测链接是否跳转到列表页（车辆已删除）
- 批量检查车辆状态

**核心函数**:
```javascript
checkVehicleStatus(page, listingUrl)
// 返回: { status: 'alive'|'sold'|'invalid'|'error', reason: string }

checkVehiclesStatus(vehicles, limit = 20)
// 批量检查，支持限制数量防封锁
```

**检测关键词**:
- `sold`, `已售`, `已售出`, `已售罄`
- `no longer available`, `unavailable`, `closed`
- `item was sold`

---

### 2. ✅ 黑名单管理系统

**文件**: `vehicle-status-check.js` + `blacklist-manager.js`

**功能**:
- 持久化存储已售/无效车辆
- 快速检查车辆是否在黑名单中
- 自动添加检测到的无效车辆

**数据结构**:
```javascript
{
  "vehicles": [
    {
      "id": "5786836792",
      "listingUrl": "https://...",
      "model": "Tiida",
      "year": 2005,
      "price": 3100,
      "reason": "链接跳转到列表页",
      "addedAt": "2026-03-03T10:00:00.000Z"
    }
  ],
  "lastUpdated": "2026-03-03T10:00:00.000Z"
}
```

**核心函数**:
```javascript
isBlacklisted(listingId)  // 快速检查
addToBlacklist(vehicle)   // 添加到黑名单
filterValidVehicles(vehicles)  // 过滤黑名单车辆
```

**使用方式**: 在评分前运行状态检查


## 🔧 使用示例

### 示例1: 快速过滤黑名单

